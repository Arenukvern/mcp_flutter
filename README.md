## ğŸš€ Getting Started (Brief Overview)

For a quick setup, please refer to the [Quick Start Guide in our documentation](./docs/getting-started/quick-start.md).

The basic steps involve:
1. Cloning the repository.
2. Installing dependencies: `make install`.
3. Adding `mcp_toolkit` to your Flutter app.
4. Initializing `MCPToolkitBinding` in your app.
5. Running your Flutter app with specific debug flags.
6. Configuring your AI tool to connect to the MCP server.
